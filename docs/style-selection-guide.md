## Tier 1 风格（强烈推荐）

### 1. Snoopy 温暖漫画

**视觉特征**：
```
Charles Schulz Peanuts 风格
- 圆头简笔小人物（Charlie Brown、Snoopy）
- 温暖米色背景（#FFF8E8）
- 简洁手绘线条
- 温暖的日落色调（橙+蓝+绿）
```

**最佳场景**：
- ✓ 品牌/产品介绍（亲切感）
- ✓ 教育培训课件（容易理解）
- ✓ 个人 IP、创业融资（创意感）
- ✓ 社区建设、用户故事

**建议用途**：
```
适合叙事性演讲，强调温暖和人性化
例："如何建立远程团队的文化"
    "创业故事和个人成长"
```

**Path 支持**：A 和 B（双路径皆可）
**AI 生成效果**：⭐⭐⭐⭐⭐（极好）

**样例**：见 `backend/static/styles/samples/style-01-snoopy.png`

---

### 2. 学習漫画 Manga

**视觉特征**：
```
日式教育漫画风格
- 夸张的表情和手势
- 惊讶、恍然大悟的脸部表情
- 速度线和星光效果
- 清晰的对话泡
```

**最佳场景**：
- ✓ 教程和培训（角色反应驱动学习）
- ✓ 知识科普（趣味性强）
- ✓ 技术讲解（复杂概念简化）
- ✓ 交互式课堂

**建议用途**：
```
3-5 面板布局，让角色的惊讶/理解表情传达学习进度
例："机器学习入门"
    "如何使用新工具"
```

**Path 支持**：A 和 B
**AI 生成效果**：⭐⭐⭐⭐⭐

---

### 3. Ligne Claire 清线漫画

**视觉特征**：
```
欧洲漫画风格（清晰线条派）
- 均匀的墨线条（无粗细变化）
- 平涂纯色块
- 零视觉噪音
- 极高信息清晰度
```

**最佳场景**：
- ✓ 产品说明（清晰指引）
- ✓ 流程图和步骤演讲
- ✓ 信息图表（数据可视化）
- ✓ 技术架构说明

**建议用途**：
```
需要精确信息传达的场景
例："API 调用流程图"
    "产品功能说明"
```

**Path 支持**：A 和 B
**AI 生成效果**：⭐⭐⭐⭐

---

### 4. Neo-Pop 新波普

**视觉特征**：
```
当代波普艺术
- 大型粗体字（标题占 50% 画面）
- 高对比鲜艳色块（粉、青、黑、金）
- 杂志感排版
- 10:1 字号对比
```

**最佳场景**：
- ✓ 年轻品牌/创意公司
- ✓ 产品发布会（视觉冲击）
- ✓ 社交媒体推广
- ✓ 时尚/科技行业

**建议用途**：
```
视觉优先，强调当代感和活力
例："新产品发布"
    "品牌重塑演讲"
```

**Path 支持**：A 和 B
**AI 生成效果**：⭐⭐⭐⭐

**注意**：避免在 Prompt 中写具体的字号数字（如"20pt"），会被渲染成真实文字。

---

### 5. Neo-Brutalism 新粗野主义

**视觉特征**：
```
粗糙主义现代风格（2024 年流行）
- 4-6px 粗黑边框（所有元素）
- 高饱和纯色块（无渐变）
- 超大字排版（3-6vw）
- 8px 偏移实色阴影
```

**最佳场景**：
- ✓ 企业内部培训（信息密集）
- ✓ 线下技术分享（远距离可读）
- ✓ 数据报告（多模块并存）
- ✓ Workshop 工作坊

**建议用途**：
```
远距离演讲，需要清晰的视觉优先级
例："130+ 页的 AI 培训课程"
    "多个数据展示的综合报告"
```

**Path 支持**：A 和 B
**特殊优势**：**纯 CSS 实现，无需 AI 图片生成，速度最快** ⚡

**AI 生成效果**：⭐⭐⭐⭐⭐（Path B）或 N/A（Path A 纯 CSS）

**成功案例**：2026-02-09 蕴煜 AI 培训项目 131 页全部成功渲染

## Tier 2 风格（推荐）

### 6. xkcd 白板手绘

简洁幽默的技术手绘。适合极客受众、课堂讲解、复杂概念秒懂。

**配色**：白底 + 黑线 + 灰色
**最佳场景**：技术分享、开发者大会
**Path 支持**：A 和 B

---

### 7. 苏联构成主义

力量感强、辨识度极高的历史风格。对角线楔形、旋转文字、几何 crescendo。

**配色**：革命红(40%) + 黑(25%) + 米白(30%)
**最佳场景**：产品发布、品牌宣言、动员会
**Path 支持**：A 和 B
**关键特点**：所有文字旋转 15-30°，无水平线

---

### 8. 敦煌壁画

东方美学、庄重诗意。卷轴叙事、飞天飘带、矿物质颜料感。

**配色**：赭石(#D4A574) + 石青 + 朱砂 + 石绿 + 金
**最佳场景**：国风品牌、文化项目、高端场合
**Path 支持**：A 和 B
**适用市场**：中国、东南亚

---

### 9. 浮世绘

日本木版画。浪潮隐喻递进、日式云纹、木版画质感。

**配色**：和纸米白 + 靛蓝 + 朱红 + 金
**最佳场景**：日本/东方市场、跨境品牌
**Path 支持**：A 和 B

---

### 10. The Oatmeal 信息图漫画

搞笑夸张、信息密度适中。大头角色、极度夸张表情、信息图式数据。

**配色**：浅灰 + 亮橙 + 深灰 + 品牌红
**最佳场景**：科普、社交传播、内部培训
**Path 支持**：A 和 B
**特点**：幽默感强，适合缓解严肃话题

---

## Tier 3 风格（可用）

### 11. 温暖叙事

人物场景生动自然，最有「人情味」。暖色插画、生活场景。

**最佳场景**：用户故事、品牌故事、社区分享

---

### 12. 孔版印刷 Risograph

双色叠印独特美学。套色错位、半色调网点、粗糙纸感。

**最佳场景**：独立品牌、创意行业、音乐相关

---

### 13. 等轴测 Isometric

2.5D 游戏世界感。Monument Valley 美学、小人物走楼梯。

**最佳场景**：科技产品、SaaS 流程、应用介绍

---

### 14. Bauhaus 包豪斯

几何=逻辑。圆/三角/方/星代表 4 步，色彩心理学意义。

**最佳场景**：设计行业、建筑项目、教育内容

---

### 15. 工程蓝图 Blueprint

精密机器隐喻。白线 on 深蓝底、工程网格、尺寸标注。

**最佳场景**：技术架构、工程方案、系统设计

---

### 16. 复古广告 Vintage Ad

乐观复古好感。美式插画、半色调、丝带横幅、美好生活。

**最佳场景**：消费品、零售品牌、怀旧营销

---

### 17. 达达拼贴 Collage

反规则、最另类。撕纸碎片、混搭字体、胶带、有序混乱。

**最佳场景**：创意行业、广告设计、艺术项目、破冰活动

---

### 18. 像素画 Pixel Art

RPG 任务隐喻。16-bit 调色板、横版世界地图、像素角色。

**最佳场景**：游戏相关、年轻群体、Gamification

## Professional Editorial 风格（高端专业）

这 6 种风格仅支持 **Path A（HTML→PPTX）**，因为精确排版和网格系统无法通过 AI 生成图片保证。

### P1. Pentagram Editorial

信息建筑派。字体即语言，网格即思想。瑞士网格、单一强调色。

**最佳场景**：行业分析、咨询报告、编辑杂志
**风格**：60% 留白 / 30% 内容 / 10% 强调色

---

### P2. Fathom Data Narrative

数据叙事风。每像素都承载信息。高信息密度+设计优雅。

**最佳场景**：数据报告、科学展示、研究汇报
**特点**：小数字(60pt+)作为视觉锚点

---

### P3. Müller-Brockmann Grid

瑞士网格风。客观性即美。数学精确、功能主义至上。

**最佳场景**：培训课件、技术架构、流程说明
**特点**：8 列数学网格，所有元素对齐

---

### P4. Build Luxury Minimal

奢侈极简风。精致的简单比复杂更难。75% 留白。

**最佳场景**：投资路演、品牌高管汇报、奢侈品
**特点**：金比例、微妙字重变化

---

### P5. Takram Speculative

日式思辨风。技术是思考的媒介。概念原型图为核心。

**最佳场景**：设计思维、产品愿景、战略规划
**特点**：柔和阴影、圆角、流程图艺术化

---

### P6. NYT Magazine Editorial ⭐ **强烈推荐**

纽约时报编辑风。新闻排版派。Georgia 衬线 + 编辑红线。

**最佳场景**：正式商务汇报、数据报告、行业分析、投资路演
**特点**：
- 5px 红色顶部规则线（签名风格）
- Georgia 衬线标题（36pt+）
- 小型大写字母 section label
- 水平规则系统贯穿
- 编辑蓝/编辑绿路径标记

**成功案例**：workflow.html 设计验证（用户反馈「纽约时报 html 风格非常好」）

## 如何选择：3 步决策法

### 第 1 步：确定演讲场景

```
□ 企业内部    → Neo-Brutalism / 学習漫画
□ 客户/投资人 → NYT Magazine / Pentagram
□ 大型会议    → Neo-Pop / 苏联构成主义
□ 社交媒体    → Neo-Pop / Snoopy
□ 教室/培训   → Neo-Brutalism / Ligne Claire
```

### 第 2 步：选择视觉调性

```
□ 专业权威  → NYT Magazine / Pentagram / Bauhaus
□ 温暖亲切  → Snoopy / 温暖叙事 / 敦煌壁画
□ 创意突破  → 达达拼贴 / Neo-Pop / 苏联构成主义
□ 简洁清晰  → Ligne Claire / xkcd / Neo-Brutalism
□ 文化底蕴  → 敦煌壁画 / 浮世绘
```

### 第 3 步：考虑实际约束

```
□ 我需要快速 (< 1分钟)        → Neo-Brutalism
□ 我需要可编辑 (Path A)       → 任何风格或 Editorial 系列
□ 我需要视觉冲击 (Path B)     → Snoopy / Neo-Pop / 苏联构成主义
□ 我需要降低成本              → Neo-Brutalism / Editorial 系列
□ 我需要全 AI 视觉            → Tier 1/2 风格 + Path B
```

## 常见搭配建议

### "我想要最安全的选择"
```
→ NYT Magazine Editorial (Path A)
   理由：权威感 100%，无法出错，适合所有正式场合
```

### "我想展示创新和 AI 能力"
```
→ Neo-Pop (Path B) 或 Snoopy (Path B)
   理由：视觉冲击力强，AI 生成效果好，显得前沿
```

### "我想最快完成"
```
→ Neo-Brutalism (Path A)
   理由：纯 CSS，无需 AI，< 1 分钟生成
```

### "我想最温暖和容易记住"
```
→ Snoopy 温暖漫画 (Path B)
   理由：最高的趣味性和传播性
```

### "我想要国风和文化底蕴"
```
→ 敦煌壁画 (Path B)
   理由：视觉独特，国际竞争力强
```
